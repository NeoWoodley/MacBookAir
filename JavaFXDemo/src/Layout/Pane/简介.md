# Pane简介
## GridPane
通常用于布局：第一列上的只读标签的输入表单和第二列上的输入字段。  
GridPane可以在行，列或单元格级别指定约束。  
例如，我们可以设置包含输入文本字段的第二列，以在窗口调整大小时调整大小。
